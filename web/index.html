The idea

AI agents re-reason every single run. Mimic learns once, then replays for free.

The same browser busywork runs hundreds of times a day: copying records between systems that don't talk, filling one form fifty times. Hand it to an AI agent and it reasons through every step from scratch on every run, so you pay full token cost each time, and it still breaks the moment the layout shifts. Mimic learns the task from one demonstration and replays it at almost no tokens. It only stops to think when something actually changed, fixes that step itself, and writes the fix back so the next run is free again.
